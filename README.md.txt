# Tipos de Proyectos que Puede Contener

Este repositorio puede incluir proyectos de diferentes áreas:

## Software
Aplicaciones completas o utilidades.

Ejemplos:

- herramientas de procesamiento de video
- automatización
- scripts de análisis

---

## Investigación aplicada

Prototipos relacionados con:

- inteligencia artificial
- ciencia de datos
- simulación
- ingeniería de software

## Documentación técnica

Incluye:

- diagramas de arquitectura
- metodologías
- análisis técnicos
- notas de investigación
